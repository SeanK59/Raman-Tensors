# Raman-Tensors
Calculates symmetry compositions in polarization resolved Raman scattering experiments

The Mathematica notebook in the project is a standalone application. One can also download the html and run it from a browser.
The same calculator can be accessed online at https://seank59.github.io/Raman-Tensors/
There is no need to ask for permission to use or to redistribute the calculator.
But please let me know if you find any errors, so that other people using the calculator don't make the same mistakes.
